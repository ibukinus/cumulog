# ネタバレ明示

ユーザーは、活動ログにネタバレが含まれるかどうかを明示できる。

## 優先度

Must

## 要件

* ネタバレなしのログとして記録できる
* 軽微なネタバレを含むログとして記録できる
* 重大なネタバレを含むログとして記録できる
* ネタバレを含むログであることを閲覧時に確認できる
* ネタバレを含む活動ログの場合、保存前にネタバレ設定を確認できる
* 未選択時は「ネタバレなし」として扱う
* MVPでは、ネタバレ有無は活動ログの性質を明示するラベルとして扱う
* ネタバレを含むログの本文非表示や折りたたみ表示は、初期リリースでは必須としない
* ネタバレなし、軽微なネタバレ、重大なネタバレの選択肢には、ユーザーが判断しやすい説明を表示する

## 受け入れ基準

* 活動ログに対して、ネタバレなし、軽微なネタバレ、重大なネタバレのいずれかを設定できる
* 未選択の場合は、ネタバレなしとして扱う
* ネタバレを含む活動ログの場合、閲覧時にネタバレを含むことをラベルとして確認できる
* ネタバレを含むログの本文非表示や折りたたみ表示は、初期リリースでは必須としない
* ネタバレなし、軽微なネタバレ、重大なネタバレの選択肢には、ユーザーが判断しやすい説明が表示される
