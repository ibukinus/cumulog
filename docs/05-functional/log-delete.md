# 活動ログ削除

ユーザーは、自分が作成した活動ログを削除できる。

## 優先度

Must

## 要件

* 自分の活動ログを削除できる
* 削除前に確認できる
* 削除前に、削除しても外部サービスや他クライアントに取得済みの情報が残る可能性があることを確認できる
* 削除後、Cumulogの活動ログ一覧に表示されなくなる
* 削除に失敗した場合、対象の活動ログはCumulog画面上から消えたように扱わない

## 受け入れ基準

* 活動ログを削除すると、Cumulogの一覧・詳細から表示されなくなる
* 削除前に、削除しても外部サービスや他クライアントに取得済みの情報が残る可能性があることを確認できる
* 削除に失敗した場合、対象ログが一覧・詳細から消えたように表示されない
