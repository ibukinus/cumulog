# 年表表示

ユーザーは、自分の活動ログを時系列で振り返れる。

## 優先度

Must

## 要件

* 活動ログを活動日の日付降順で確認できる
* 同一活動日の活動ログは、作成日時の降順で確認できる
* 過去の活動を振り返れる
* 未来日の活動ログも一覧上で確認できる
* 活動の積み重ねを確認できる
* 自分の活動履歴として一覧できる

MVPでは、活動日を基準にした日付降順の一覧表示を年表表示として扱う。

月別・年別のグルーピング表示は、初期リリースでは後回し可能とする。

未来日の活動日も登録できるが、MVPでは予定管理、通知、リマインダー、カレンダー同期は対象外とする。

## 受け入れ基準

年表表示はMVPでは日付降順の一覧表示として実現するため、受け入れ基準は[活動ログ一覧](log-list.md)の受け入れ基準に含まれる。
