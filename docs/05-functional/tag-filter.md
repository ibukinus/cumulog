# タグによる絞り込み

ユーザーは、タグを使って活動ログを整理できる。

## 優先度

Must

## 要件

* タグで活動ログを絞り込める
* タグは自由入力で登録できる
* 1つの活動ログに複数のタグを登録できる
* 1つの活動ログ内で同一タグを重複登録しない
* 空のタグは登録できない
* MVPでは、1つのタグを選択し、そのタグが登録された自分の活動ログのみを表示できる
* MVPでは、タグ文字列の完全一致で絞り込む
* MVPではタグの表記ゆれ統合は行わない
* MVPではタグの部分一致検索は行わない
* MVPでは複数タグによるAND / OR絞り込みは行わない

MVPでは、全文検索、高度な検索条件、保存検索は対象外とする。

## 受け入れ基準

* 任意のタグを1つ選択すると、そのタグが付与された自分の活動ログのみを一覧で確認できる
* タグ絞り込みは、MVPではタグ文字列の完全一致で行われる
* タグの表記ゆれ統合、部分一致検索、複数タグによるAND / OR絞り込みはMVPでは行われない
* 1つの活動ログ内で同一タグを重複登録できない
* 空のタグを登録できない
