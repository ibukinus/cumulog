# 月次振り返り

ユーザーは、月ごとの活動を振り返れる。

## 優先度

Should

MVPでは、月次振り返りはShouldとし、初期リリースでは後回し可能とする。

## 要件

* 月ごとの活動ログを確認できる
* 月内に行った活動の概要を確認できる
* 月ごとの活動量を確認できる
