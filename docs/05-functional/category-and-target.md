# 活動種別・対象名による整理

ユーザーは、活動種別や対象名を活動ログに登録できる。

## 優先度

* 活動種別・対象名の登録：Must
* 活動種別・対象名による絞り込み：Should

## 要件

* 活動種別を任意で登録できる
* 活動種別はMVPでは自由入力として扱う
* 対象名を任意で登録できる
* 対象名はMVPでは1件のみ登録できる
* 対象名は、作品名、イベント名、人物名、プロジェクト名などを想定する
* 対象名の複数登録は将来検討とする

MVPでは、活動種別や対象名による絞り込みは後回し可能とする。

## 受け入れ基準

* 活動ログに活動種別・対象名を任意で登録でき、一覧・詳細で確認できる（[活動ログ一覧](log-list.md)・[活動ログ詳細](log-detail.md)の受け入れ基準に含まれる）
