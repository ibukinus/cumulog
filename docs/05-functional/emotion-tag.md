# 感情タグ

ユーザーは、活動ログに感情タグを付けられる。

## 優先度

Should

MVPでは、感情タグはShouldとし、初期リリースでは後回し可能とする。

## 要件

* 活動に対する感情をタグとして登録できる
* 感情タグで活動ログを振り返れる
* 感情タグは通常のタグやメモとは別の情報として扱える

## 受け入れ基準

* 感情タグを実装する場合、活動ログに感情タグを登録でき、通常のタグとは別の情報として確認できる
* 感情タグが初期リリースで未実装でも、MVP成立条件には影響しない
