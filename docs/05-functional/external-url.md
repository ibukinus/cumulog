# 外部URL登録

ユーザーは、活動ログに外部URLを紐づけられる。

## 優先度

Must

## 要件

* note記事へのURLを登録できる
* ブログ記事へのURLを登録できる
* SNS投稿へのURLを登録できる
* YouTubeなどの動画ページへのURLを登録できる
* 公式ページへのURLを登録できる
* 1つの活動ログに複数の外部URLを登録できる
* MVPでは、外部URLとして http または https のURLを登録対象とする
* 登録したURLから外部ページを開ける
* 外部URL先の内容、安全性、公開状態はCumulogでは保証しない
* 登録されたURLが無効になった場合でも、活動ログ自体は保持される
* 登録できないURL形式の場合、保存前にエラーとして確認できる

MVPでは、URLの内容取得、本文保存、アーカイブ、プレビュー生成、リンク切れ検出は対象外とする。

## 受け入れ基準

* 活動ログに複数の外部URLを登録できる
* MVPでは、http または https のURLを登録できる
* 登録できないURL形式の場合、保存前にエラーとして確認できる
* 登録された外部URLを選択すると、外部ページを開ける
* 外部URL先の本文保存、アーカイブ、プレビュー生成、リンク切れ検出はMVPでは行わない
* 外部URL先の内容、安全性、公開状態はCumulogでは保証しない
