# 活動ログ詳細

ユーザーは、活動ログの詳細を確認できる。

## 優先度

Must

## 要件

* 活動ログの内容を確認できる
* 活動日を確認できる
* 活動種別を確認できる
* 対象名を確認できる
* タグを確認できる
* メモを確認できる
* 外部URLを確認できる
* ネタバレ有無を確認できる
* AT Protocol上の公開レコードとして保存されていることを確認できる
* 活動ログ詳細の取得に失敗した場合、対象ログが存在しない状態とは区別して表示する

## 受け入れ基準

* 一覧から活動ログを選択すると、登録した内容を詳細画面で確認できる
* 詳細画面では、活動ログがAT Protocol上の公開レコードとして保存されていることを確認できる
* 詳細取得に失敗した場合、対象ログが存在しない状態とは区別して表示される
