# 活動ログ編集

ユーザーは、自分が作成した活動ログを編集できる。

## 優先度

Must

## 要件

* 自分の活動ログの内容を修正できる
* タイトルを修正できる
* 活動日を修正できる
* 活動種別を修正できる
* 対象名を修正できる
* タグを修正できる
* 外部URLを修正できる
* メモを修正できる
* ネタバレ有無を修正できる
* 編集後の内容も公開データとして扱われることをユーザーに示せる
* 編集に失敗した場合、編集が成功したように表示しない
* 編集に失敗した場合、入力済み内容を可能な限り保持し、ユーザーが再試行できるようにする

## 受け入れ基準

* 既存の活動ログを編集して保存すると、再読み込み後も編集後の内容が表示される
* 編集画面では、編集後の内容も公開データとして扱われる旨を確認できる
* 編集に失敗した場合、編集が成功したように表示されない
* 編集に失敗した場合、入力済み内容が可能な限り保持され、ユーザーが再試行できる
