# 公開共有ページ

ユーザーは、自分の活動ログを指す公開URLを共有でき、そのURLを受け取った人は未認証で該当の活動ログを閲覧できる。

## 優先度

Should

Bluesky共有投稿（[Bluesky共有](bluesky-share.md)）とあわせて価値を持つ機能であり、同じ優先度とする。

## 要件

* 単一の活動ログを未認証で閲覧できる公開ページを提供する
* 公開ページのURLは、活動ログの所有者（DID）とレコード（rkey）から構成する
* 公開ページは閲覧専用とし、作成・編集・削除などの操作は提供しない
* 活動ログがAT Protocol上の公開データであることを公開ページ上に表示する
* 存在しない、削除済み、または読み込めない活動ログのURLでは、閲覧できない旨を表示する
* Bluesky共有投稿の文面に、公開ページのURLを含められる
* 他ユーザーの活動ログの探索・一覧・検索機能は提供しない（公開ページは共有URLを知っている場合の単一ログ閲覧に限る）

## 受け入れ基準

* 未認証のブラウザで、共有URLから活動ログの内容を閲覧できる
* 公開ページから活動ログの作成・編集・削除の操作はできない
* 削除済み・存在しない活動ログの共有URLでは、閲覧できない旨が表示される
* Bluesky共有投稿の既定文面に公開ページのURLが含まれる
