# 活動ログ一覧

ユーザーは、自分が作成した活動ログを一覧で確認できる。

## 優先度

Must

## 要件

* 自分の活動ログを一覧で表示できる
* 活動日を確認できる
* 活動タイトルを確認できる
* 活動種別を確認できる
* タグを確認できる
* 対象名を確認できる
* 外部URLの有無を確認できる
* ネタバレ有無を確認できる
* 活動ログを活動日の日付降順で確認できる
* 同一活動日の活動ログは、作成日時の降順で表示する
* 活動ログ一覧の取得に失敗した場合、活動ログが存在しない状態とは区別して表示する

## 受け入れ基準

* 作成済みの自分の活動ログが、活動日の日付降順で一覧表示される
* 同一活動日の活動ログは、作成日時の降順で表示される
* 一覧では、活動日、タイトル、タグ、対象名、活動種別、外部URLの有無、ネタバレ有無を確認できる
* 一覧取得に失敗した場合、活動ログが存在しない状態とは区別して表示される
