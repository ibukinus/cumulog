# 初回利用

ユーザーは、サービスの目的を理解した上で利用を開始できる。

## 優先度

Must

## 要件

* Cumulogが公開活動ログサービスであることを理解できる
* 活動ログがAT Protocol上の公開データとして保存されることを理解できる
* MVPではBluesky公式PDSを利用するAT Protocolアカウントを主な対応対象とすることを理解できる
* 初回利用時に迷わず最初の活動ログを作成できる
* 複雑なチュートリアルはMVPでは対象外とする
