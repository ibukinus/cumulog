# 初回利用

ユーザーは、サービスの目的を理解した上で利用を開始できる。

## 優先度

Must

## 要件

* Cumulogが公開活動ログサービスであることを理解できる
* 活動ログがAT Protocol上の公開データとして保存されることを理解できる
* MVPではBluesky公式PDSを利用するAT Protocolアカウントを主な対応対象とすることを理解できる
* 利用開始前に利用規約とプライバシーポリシーを確認できる
* 初回利用時に迷わず最初の活動ログを作成できる
* 複雑なチュートリアルはMVPでは対象外とする

利用規約・プライバシーポリシーの文面と提示方法は基本設計以降で定義する。

## 受け入れ基準

* 初回利用時に、Cumulogが公開活動ログサービスであり、活動ログがAT Protocol上の公開データとして保存されることを確認できる
* 利用開始前に、利用規約とプライバシーポリシーを確認できる
* 利用開始後、追加の説明を探すことなく最初の活動ログの作成に到達できる
