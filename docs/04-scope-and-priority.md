# スコープと優先度

MVPのスコープは、MoSCoWの優先度分類で一元的に定義する。個別の機能要件の詳細は[機能要件](05-functional/index.md)を参照。

## 優先度の定義

### Must

MVPで必ず実現する機能または要件。

Mustが満たされない場合、CumulogのMVPは成立しない。

### Should

MVPの価値を高めるが、初期リリースで未実装でもMVP成立条件を満たせる機能。

Shouldは、初期リリース後に優先的に検討する。

### Could

初期リリース後に検討する機能。

MVPの成立には影響しない。

### Won't

MVPでは実施しない機能。

## Must — MVPで必ず実現する

* Bluesky公式PDSを利用するAT Protocolアカウントで利用できる
* 任意PDS対応はMVPでは対象外とする
* 活動ログをAT Protocol上のCumulog独自レコードとして作成できる
* 活動ログの正本をAT Protocol上のCumulog独自レコードとして扱える
* 自分の活動ログレコードを一覧で見られる
* 活動ログレコードの詳細を見られる
* 自分の活動ログレコードを編集できる
* 自分の活動ログレコードを削除できる
* 活動ログが公開データとして保存されることをユーザーに示せる
* 活動ログ保存前に保存内容を確認できる
* 編集後の内容も公開データとして扱われることをユーザーに示せる
* 削除前に、削除しても外部に残る可能性があることを示せる
* 活動ログを日付降順で振り返れる
* 同一活動日の活動ログは作成日時降順で表示できる
* 未来日の活動日を登録できる
* 活動ログに外部URLを複数添付できる
* 外部URLは http / https を登録対象とする
* 活動ログにタグを付けられる
* タグで活動ログを絞り込める
* MVPでは単一タグの完全一致で絞り込める
* 1つの活動ログ内で同一タグを重複登録しない
* 空のタグを登録できない
* 活動ログに対象名を登録できる
* 活動ログに活動種別を登録できる
* 活動ログに短文メモを登録できる
* 活動ログにネタバレ有無を設定できる
* ネタバレ有無はMVPではラベルとして表示できる
* ネタバレ選択肢にはユーザーが判断しやすい説明を表示できる
* 初回利用時に、公開活動ログサービスであることを理解して利用を開始できる
* Cumulog画面上では自分の活動ログのみを表示・操作対象にする
* 認証失効、権限不足、保存失敗、編集失敗、削除失敗、一覧取得失敗をユーザーに分かる形で表示できる

MVPでは、活動日を基準にした日付降順の一覧表示を年表表示として扱う。

未来日の活動日も登録できるが、予定管理機能は対象外とする。

## Should — MVPに含めたいが後回し可能

* Blueskyに共有投稿できる
* Bluesky投稿前に投稿内容を確認できる
* 活動種別で絞り込める
* 対象名で絞り込める
* 年月で絞り込める
* 感情タグを付けられる
* 月別に活動ログを確認できる

## Could — 初期リリース後に検討する

* 年間まとめを表示できる
* 同じ対象に関する他人の公開ログを見られる
* カスタムフィードを提供する
* URLのプレビューを表示する
* ログをMarkdownやJSONでエクスポートできる
* 画像添付に対応する
* カレンダー表示に対応する
* 創作・開発ログ向けテンプレートを提供する
* 通知やリマインダーを提供する
* 外部サービスとの連携を拡張する
* 活動日を日時で登録できる
* 対象名を複数登録できる
* 任意PDSに対応する
* タグの部分一致検索に対応する
* 複数タグによるAND / OR絞り込みに対応する
* タグの表記ゆれ統合に対応する

## Won't — MVPでは実施しない

* 任意PDS対応
* 非公開ログ
* 下書き
* 限定公開
* 複雑な公開範囲設定
* 長文エディタ
* 日時の登録
* 課金
* 投げ銭
* DM
* フォロワー管理
* いいね、コメント、リポストなどのSNS的交流機能
* 大規模コミュニティ機能
* 外部サービスからの自動インポート
* AI要約
* AIレコメンド
* 公式データベースの整備
* 全ジャンルに最適化された詳細項目
* 高度な統計分析
* 画像・動画の本格ホスティング
* URL先本文の保存・アーカイブ
* URLプレビュー生成
* リンク切れ検出
* 高度な検索機能
* 全文検索
* 保存検索
* 複数タグによるAND / OR絞り込み
* タグの部分一致検索
* タグの表記ゆれ統合
* 大規模な他ユーザー探索機能
* カスタムフィード
* 他ユーザーの公開ログ閲覧
* 専用エクスポート機能
* リマインダー
* 通知
* カレンダー同期
* 予定の自動更新

Won'tはMVP期間中に実施しないことを示す。同じ項目がCouldにも挙がっている場合（カスタムフィード、エクスポート、タグ検索の拡張など）は、初期リリース後に改めて検討する。
